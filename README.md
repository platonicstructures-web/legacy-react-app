# Modernized Legacy App

This application was successfully migrated from C++ to React using **LegacyForge**.

## Run and Deploy

You can run this full-stack application instantly in the cloud using Google's Project IDX.

[![Open in Project IDX](https://idx.google.com/run_badge.svg)](https://idx.google.com/import?url=https://github.com/platonicstructures-web/legacy-react-app)

## Local Development

1. Clone the repository
2. Run `npm install`
3. Run `npm run dev`
